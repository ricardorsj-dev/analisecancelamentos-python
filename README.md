# 📊 Análise de Cancelamento de Clientes (Churn Analysis)

Este projeto tem como objetivo analisar os fatores que influenciam o cancelamento de clientes em uma base de dados bancária.

Através de Python e análise exploratória de dados (EDA), foram identificados padrões de comportamento que ajudam a entender por que clientes deixam de utilizar os serviços.

---

## 🚀 Tecnologias utilizadas

- Python
- Pandas
- Plotly

---

## 📂 Estrutura do projeto

- main.py → Script principal de análise
- ClientesBanco.csv → Base de dados
- insights.txt → Principais conclusões da análise

---

## 🔍 Etapas da análise

1. Importação e leitura dos dados
2. Tratamento de dados ausentes
3. Análise estatística descritiva
4. Criação de gráficos interativos
5. Identificação de padrões de cancelamento

---

## 📊 Principais insights

- Clientes com maior número de contatos tendem a cancelar mais
- Clientes com menor limite apresentam maior taxa de cancelamento
- Certas categorias de cartão possuem maior retenção
- Faixa salarial influencia diretamente na permanência do cliente

---

## 🎯 Objetivo

Transformar dados em decisões estratégicas para retenção de clientes.

---

## 💡 Possíveis melhorias

- Modelos de Machine Learning para prever churn
- Dashboard interativo
- Análise de correlação mais profunda
